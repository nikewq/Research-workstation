# 博士工作台 · PhD Workstation

> 一个为博士研究生定制的全功能工作台，集成日程、文献、实验记录、反思日记、留言墙等模块。

## 在线访问

🔗 [GitHub Pages](https://nikewq.github.io/Research-workstation/)

## 主要功能

- **甘特图** — 可视化项目时间线，支持多项目管理
- **反思日记** — 每日记录，历史数据持久保存
- **留言墙** — 社区互动，AI 自动回复
- **文献管理** — 论文追踪与笔记
- **实验日志** — 记录 GPU 实验进展
- **界面主题** — 10+ 主题（含武侠、可爱、大女主、各高校 VI 配色）

## 高校主题

内置 32 所高校官方 VI 标准色，支持 AI 生成任意学校配色：

清华大学 · 北京大学 · 复旦大学 · 上海交通大学 · 浙江大学 · 南方科技大学 · 武汉大学 · 中山大学 等

## 本地开发

直接打开 `index.html` 即可，无需构建。

## 技术栈

纯原生 HTML/CSS/JS · 单文件 · 无依赖
